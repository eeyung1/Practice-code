# Exercise 1: Simple Prompt Writing

---

## Objective
Write prompts using different styles and observe how the model responds to each type.

---

## Prompt 1: Direct Instruction Style

**Prompt:**
> "List three interpreted languages"

**Model Output:**
Interpreted languages execute code line-by-line using an interpreter at runtime rather than pre-compiling the entire program into machine-level instructions. This provides cross-platform portability and faster development cycles.

Three common interpreted languages include:
* Python: Highly popular for its simple syntax, it is widely used in data science, artificial intelligence, and web development.
* JavaScript: The essential language for interactive web development, interpreted directly by web browsers to run client-side scripts.
* Ruby: An object-oriented language designed for programmer productivity, frequently used for building web applications via the Ruby on Rails framework.

## Observation
The tested promptwed the AI to explain what interpreted languages are and how it works.

---

## Prompt 2: Question Style

**Prompt:**
> "What are three interpreted languages?"

**Model Output:**
Interpreted languages are executed line-by-line at runtime by an interpreter, skipping the need for a separate compilation step. This allows for faster development cycles and greater platform independence, though they generally run slower than compiled languages.

Three common interpreted languages are:
* Python: A versatile language used for data science, AI, and automation.
* JavaScript: The standard for interactive web development.
* Ruby: Designed for productivity and simplicity, often used in web applications.

## Observation
This prompt stated clearly how interpreted languages generally run slower than compiled languages

---

## Comparison and Analysis

The key differences I noticed was that in direct instruction, it's more formal and academic while the question style is more conversational and accessible. Both responses provided the same three languages (Python, JavaScript, Ruby) with definitions, but the direct instruction style used more detailed, comprehensive descriptions while the question style was more concise and to-the-point.
